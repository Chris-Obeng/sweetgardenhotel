# Sweet Garden Hotel

A presentation-first luxury hotel website for Sweet Garden Hotel in Kumasi, Ghana.

## Commands

- `pnpm install` - install dependencies
- `pnpm dev` - start the Vite dev server on port `23857`
- `pnpm typecheck` - run TypeScript checks
- `pnpm build` - create the production build

## Project

- `src/components/sections/` - homepage sections
- `src/components/ui/` - custom hotel UI primitives
- `src/lib/constants.ts` - hotel content and data
- `src/index.css` - typography, color system, and global styles

The site is intentionally frontend-only. Backend/API workspace packages and Replit-only scaffolding are not required.
